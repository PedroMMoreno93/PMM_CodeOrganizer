# PMM_CodeOrganizer

Xcode Source Editor Extension to organize Swift files:
- Sort imports
- Insert // MARK: sections
- (Optional) Reorder members and top-level declarations
- Dry-run suggestions

## Install (dev)
- Open the Xcode project
- Run the extension scheme with Xcode.app as host
- Enable the extension in macOS Settings > Privacy & Security > Extensions > Xcode Source Editor
